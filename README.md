# Amazon-sales-Dashboard
Interactive Amazon Sales Performance Dashboard built using Microsoft Power BI to analyze sales, profit, and product performance across different regions and categories. The dashboard provides key business insights using data modeling, DAX measures, and visualizations to support data-driven decision making.


This project presents an interactive sales analytics dashboard developed using Microsoft Power BI. The dashboard helps analyze sales performance, profit trends, and product category insights for better business understanding.

Key Features

Sales analysis by region and category

Profit and revenue performance tracking

Monthly sales trend analysis

Top-selling products identification

Interactive filters and KPIs

Tools Used
SQL
Power BI
DAX
Data Modeling
Data Visualization
